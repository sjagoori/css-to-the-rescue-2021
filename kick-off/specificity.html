Specificity

Info & voorbeelden van: https://specifishity.com/

Specificity beschrijft hoe "belangrijk" css een selector vindt.

  1. Specificity bestaat uit 3 waardes: X-Y-Z
  2. De hoeveelheid ID selectors: X-0-0
  3. De hoeveelheid class en attribute selectors & pseudo-classes: 0-Y-0
  4. De hoeveelheid element selectors & pseudo-elementen: 0-0-Z
